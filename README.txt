SIA - Painel Editável na Tela

Tudo que importa é editável DIRETO NO SITE, sem mexer em código.

COMO USAR
---------
1. Abra o arquivo index.html no navegador (Chrome, Edge, etc.).
2. Clique sobre:
   - Nome do aluno
   - Curso
   - Matrícula
   - CPF
   - Textos de situação, pendências, declaração
   - Células da tabela de disciplinas

   Todos esses campos têm contenteditable="true".
   É só clicar e digitar o texto que você quiser.

3. Para adicionar disciplinas:
   - Clique em "+ Adicionar disciplina" (abaixo do título "Relação de disciplinas integralizadas").
   - Uma nova linha aparecerá com "COD", "Nome da disciplina" e "Aprovado".
   - Clique em cada célula e troque pelo código, nome e observação que quiser.

4. Para remover a última disciplina:
   - Clique em "Remover última linha".
   - A última linha da tabela será apagada (sempre deixa pelo menos 1).

5. Depois de preencher tudo:
   - Use Ctrl + P (Windows) ou Cmd + P (Mac) para imprimir ou salvar em PDF.
   - Você também pode usar a ferramenta de captura de tela para tirar um print.

NÃO PRECISA EDITAR CÓDIGO.

ARQUIVOS
--------
- index.html  -> estrutura da tela + script para adicionar/remover linha
- styles.css  -> visual profissional estilo sistema acadêmico
