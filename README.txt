SIA - Painel de Situação do Aluno (Layout profissional, edição na tela)

O QUE VOCÊ TEM AQUI
-------------------
- Layout claro, neutro e profissional (sem neon).
- Uma única tela com:
  - Dados do aluno (nome, matrícula, CPF)
  - Situação acadêmica / status para colação / observações
  - Texto de pendências na matriz
  - Declaração de aptidão à colação de grau
  - Tabela de disciplinas (código, nome, observação)
- Todos os campos importantes são editáveis DIRETO NO NAVEGADOR (contenteditable).

COMO USAR
---------
1. Abra o arquivo index.html no navegador (Chrome, Edge, etc.).

2. Clique e digite nos campos que já vêm preenchidos de exemplo:
   - Iniciais do aluno (bolinha à esquerda)
   - Nome do aluno
   - Curso
   - Matrícula
   - CPF
   - Textos de situação acadêmica, status para colação e observações
   - Texto de pendências na matriz
   - Título e parágrafos da declaração de aptidão

3. Na tabela de disciplinas (parte de baixo da tela):
   - Cada célula é editável (código, nome, observação).
   - Use os botões:
       "+ Adicionar disciplina"  -> cria uma nova linha
       "Remover última linha"    -> apaga a última linha (sempre deixa pelo menos uma)

4. Depois de preencher tudo:
   - Use Ctrl + P (Windows) ou Cmd + P (Mac) para imprimir ou salvar em PDF.
   - Você também pode tirar print de tela normalmente.

ARQUIVOS
--------
- index.html  -> estrutura da página + script simples para adicionar/remover linhas.
- styles.css  -> estilos (cores, fontes, espaçamento).

Não é necessário mexer em código para usar no dia a dia:
basta abrir, preencher, imprimir e mostrar para o aluno.
